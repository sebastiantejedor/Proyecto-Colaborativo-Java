# Proyecto-Colaborativo-Java
Actividad Colaborativa haciendo uso de Github

# Observaciones Importantes:
- Uso de JDK 26 (Ultima version) Uso de las características modernas del lenguaje Java.
- Antes de crear un Pull Request, verificar que el código compile correctamente.
- Mantener una estructura de código limpia y legible. 
- Uso de Funciones con parametros para cada Funcionalidad.
- Cada desarrollador debe trabajar en una rama independiente, La rama main se encuentra protegida por el Integrador.
- No realizar cambios directamente sobre la rama main.
- Todos los cambios deben integrarse mediante Pull Request.
